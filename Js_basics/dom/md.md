Markdown syntax includes headings, emphasis, lists, links, and code blocks, forming the core for readable documents.

## Headings
Create hierarchy with hash symbols (#).
- `# Heading 1` for top level.
- `## Heading 2`, `### Heading 3` (up to ######).
- Add space after hashes; blank line before for separation.[2]

## Emphasis
Format text for stress or boldness.
- `*italic*` or `_italic_`.
- `**bold**` or `__bold__`.
- `***bold italic***` for combined effect.[2]

## Lists
Organize content with bullets or numbers.
- Unordered: `- Item` or `* Item` (indent 2-4 spaces for nesting).
- Ordered: `1. Item`, `2. Item`.
- Task lists (GFM): `- [x] Done`.[2]

## Links and Images
Connect to resources visually.
- Links: `[text](url)` or `[text](url "title")`.
- Images: `![alt](image.jpg)`.
- Reference style: `[text][id]` with `[id]: url` at bottom.[2]

## Code Blocks
Display code cleanly.
- Inline: `` `code` ``.
- Block: Triple backticks (```) with optional language: ```` ```html\n<code>\n``` ```.
- Indent 4 spaces as alternative [].

## Additional Essentials
- Blockquotes: `> Quote`.
- Horizontal rule: `---` or `***`.
- Tables: `| Col1 | Col2 | \n|------|------| \n| Val  | Val  |`.
- Strikethrough: `~~text~~` (GFM) [].
